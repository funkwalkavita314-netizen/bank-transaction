# Bank-Transaction System

## Project Description
This is a simple python program that simulates basic bank transactions like deposit, withdraw , bank balance , adding new accountholder , edit account data , delete account holder, amount transfer and all.

The project is connected with MySQL database to store and manage customer account data permanently.

## Why I created This Project
I created this project to practise:
- python functions
- Modiles
- Conditional statements
- User input handling
- MySQL database connection
- Basic banking logic

## Features
- Created account
- Deposit money
- Withdrawl money
- Check balance
- transfer amount from one account to another
- delete account
- edit data
- Store account data in MySQL database

## Requirements
- Python 3.x
- MySQL
- mysql-connector-python-library

## How to Run
1. Download the project
2. Install required library:
    pip install mysql-connector-python
3. Make sure MySQL is running and database is created
4. Open terminal in the project folder
5. Run:
python Main.py
